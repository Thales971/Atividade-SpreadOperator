# Atividade Spread Operator

Este repositório contém uma coleção de exercícios em JavaScript voltados ao uso do **Spread Operator (`...`)**, parte das funcionalidades introduzidas no ES6 e posteriores.

## Objetivo

Praticar a manipulação de arrays e objetos de forma imutável, evitando efeitos colaterais e respeitando o princípio da imutabilidade — uma prática essencial em aplicações modernas (como React, Vue, etc.).

## Conteúdo

O arquivo principal `atividade-new-js.js` traz:

1. Copiando arrays com spread
2. Adicionando itens sem alterar o original
3. Combinando dois arrays
4. Cópia segura e explicação sobre mutação
5. Atualização de dados em objetos
6. Adição dinâmica de propriedades em objetos
7. Exercício de carrinho de compras
8. Mesclagem de listas diversas
9. Uso de spread em funções e comportamentos esperados em casos de erro
10. Exercícios avançados de imutabilidade (atualização de perfil, configuração de sistema, lógica de jogo da velha)

Cada seção contém o enunciado, implementação e exemplificação via `console.log`.

## Como executar

Certifique-se de ter o [Node.js](https://nodejs.org/) instalado e rode:

```bash
node atividade-new-js.js
```

Isso exibirá no terminal os resultados de cada exercício.

## Por que usar o Spread Operator?

O operador spread torna mais simples trabalhar com dados sem alterar a estrutura original — fundamental para manter o estado previsível em aplicações reativas. É amplamente utilizado em frameworks modernos para atualizar estados, concatenar listas e clonar objetos.
